# Instructions
- `make` (to generate executables)
- `./prob1` (run problem-1)
- `./prob2` (run problem-2)
- `./prob3 size count` (run problem-3, size=input matrix size, count=number of threads to be used)
- `make clean` (to remove all executable files)

# Language
- gcc 9.2.1 (problem 1,2)
- g++ 9.2.1 (problem 3)
